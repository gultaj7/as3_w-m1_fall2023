
# FLASHCARDS-APP

### to start the app:
```javascript
cd flashcards-app
npm start
```

### in case you get
```javascript
sh: react-scripts: command not found
```
then run: 
```javascript
npm install react-scripts --save-dev
```
### to start the json server:
```javascript
cd flashcards-app
npx json-server --watch data/db.json --port 8000
```
note: port 8000 is important detail for this project.
